Search By Phone
Type in the phone number without spaces or dashes including the area code. This is probably the easiest way to find a user while speaking over the phone. Ex: 8008675309 or 18008675309

Wildcards
Use asterisk(s) as wildcards to perform broader searches. For example, if a user has stored "John H. Doe" on their account you will not be able to find them when searching for "John Doe". Instead, try searching for... John*Doe.

If a person has a long or complicated name you can perform a partial search like... Marry Superfragilistic*

Search By Company
If you want to search by company name you must add the prefix C: to the search term. For example, do a search for C:Acme* if you want to find Acme Inc. or Acme.com.

By UserID
If you know the UserID of the account just type in the number preceded by the letter U, such as U123456.

Order Screen
Type in an order number with or without the preceding hash-code. Ex: O123456 or C057-123456. If you leave off the hash-code, you must include the prefix O (as in Order).

Project Screen
Project Numbers begin with the letter P without any spaces or dashes. Such as P7654321

Proofing Screen
Artworks begin with the letter A and the number matches the Project Number. Ex: A7654321
