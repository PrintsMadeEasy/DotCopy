Locate a Similar Design
{PRODUCT_NAME}
This feature copies all layers from the source template onto the destination product. This can speed up the time that it takes to create a matching design.

After adjusting layers sizes and positions on the destination template, you may need to replace images using the clipboard if the DPI is not sufficient.

Templates are automatically created for the destination product within the search engine. A single keyword is associated with the template using the the following format.
("TS" or "TC) + "Originating Template ID"
The prefix says whether it was from an auto-transfered Template Category, or the Search Engine. This can be helpful to locate a matching design within another browser window in the future.
