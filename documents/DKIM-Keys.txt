DKIM 1024 bit keys generated 12/10/2010

How to create on ServerBeach Proxy:

CMD
cd C:\Program Files\Apache Software Foundation\Apache2.2\bin
openssl genrsa -out privdomainkey.priv 1024
openssl rsa -in privdomainkey.priv -out publickey.pub -pubout -outform PEM

Then copy & paste private keys from files privdomainkey.priv and publickey.pub
The keys may look the same at first, but are different !




To activate DKIM with a proxy email domain, add the following lines to config.txt:

dkim=on;
dkimselector=eagle;

Example is Vinylbanners.com. If no dkim-selector is defined, DKIM is not active.
